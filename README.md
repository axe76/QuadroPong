# QuadroPong


## Cloning
```bash
$ git clone https://github.com/axe76/QuadroPong.git
```

## Dependencies
```bash
$ pip3 install -r requirements.txt
```

## Demonstration
For AI:
```bash
$ python3 neat_quadro_paddle.py
```
